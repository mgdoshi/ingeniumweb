FAQ/Glossary :

Is there a need to install any software on our computers for using Equipe ?
No. The only requirement for using Equipe is a Windows-based PC with Internet Connection with Netscape Navigator 4.5+ or IE 4+ browser. An Internet connection is a must.

If this is the case then, how does it function ?
Equipe is installed on Ingenium's web-site. You connect to our web-site with a special access which is granted to you. After you get connected to our web-site, you can use Equipe just like any other application on your machine.

Where the data is stored and who maintains it ?
All the data for your company is stored on Ingenium's database servers. The data is wholly owned by you. We at Ingenium maintain and administer the databases.

Can anybody in the world with an Internet connection look into our data ?
No. You will be granted a special access to Equipe. With this special access, you can decide :

  • Which new user to be created.
  • What access rights to be granted to the created user.
These users can then connect in the same manner and use Equipe. 'Guest' users are not permitted at all.

How many users can I create ? Is there any limit ?
You can create as many users as you need. There is no limit for it nor any extra charge on it, whatsoever.

What is meant by 'Browser-based Application' and 'Fine-Grain security Enabled' ?
'Browser-based Application' in simple terms means you only need an Internet browser like Netscape Navigator or Internet Explorer to use the application.

'Fine-grain Security Enabled' means the administrator of the Application can control and grant access to :

  • Programs used by a particular user.
  • Properties like Visible/Updateable/Insertable/Deletable of Fields, buttons or in general objects in any program.
Apart from these basic control record-level security can also be defined and controlled. For example :
  • A dealer can see only orders and other details pertaining to his/her orders.
  • A regional branch can see only relevant details which pertain to the dealers/distributors for which it is responsible and so on.
  • In case company deals in customised items for a particular client, then such an item can be made to be visible to only that particular client etc.
Which MIS Reports are present ? What about integrating this application to our already existing applications ?
Basic MIS reports like Pending orders, dealer-wise statistics, product-wise statistics etc are present. Customised reports can be easily programmed into the system.

We provide you with the facility of data export from Equipe. You can download this data the way you want into any existing application.

Can Equipe be customised to our needs ?
Due to its modular approach while designing, Equipe can be easily customised to your requirements.
